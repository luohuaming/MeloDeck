Privacy

Designed for private listening.

MeloDeck is currently in active development. This page describes the intended privacy boundaries for the product and will be updated before public distribution if implementation details or service providers change.

Music library data

MeloDeck indexes audio files and directories that the user explicitly selects. Library metadata, favorites, queue state and playback position are intended to remain in the app's local database.

The app does not claim ownership of user files and does not make them publicly available.

Cloud source access

Supported cloud sources are connected through official APIs or standard protocols. Access is limited to the account and directories selected by the user.

  • Credentials and refresh tokens are intended to be stored using iOS Keychain or Android Keystore-protected storage.
  • Temporary playback URLs and authorization headers are not stored in track records.
  • MeloDeck does not provide remote delete, public sharing or file-transfer proxy services.

Metadata enrichment

When optional metadata enrichment is enabled, MeloDeck may contact services such as AcoustID, MusicBrainz and the Cover Art Archive.

  • Acoustic fingerprint and duration may be sent for identification where licensing permits.
  • Music identifiers and metadata queries may be sent to retrieve missing titles, artists, albums or release information.
  • Raw audio files are not uploaded for metadata matching.
  • Online results only fill missing fields and do not overwrite user edits or embedded tags without user action.

Playback cache

Cloud audio may be cached locally to improve playback. Ordinary cache entries are intended to follow a user-configurable size limit and least-recently-used cleanup policy. Files explicitly marked for offline use are not automatically removed by ordinary cache cleanup.

Analytics and advertising

MeloDeck does not currently operate an advertising network or sell personal listening data. Any future diagnostics or analytics will be documented here before public release and will follow platform consent requirements.

Contact

Questions about MeloDeck or this privacy statement can be raised through the public MeloDeck project page.
