# MeloDeck

Public product and privacy pages for **MeloDeck**, a privacy-conscious music library and player for user-owned audio.

- Product page: https://luohuaming.github.io/MeloDeck/
- Privacy information: https://luohuaming.github.io/MeloDeck/privacy.html

> This repository contains only the public website. The MeloDeck application source code, credentials, provider configuration and private development documents are not included.
